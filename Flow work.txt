Example .gitignore Rules

Here are some common patterns you might include in a .gitignore file:

Ignore all log files: *.log

Ignore a specific directory: temp/

Ignore all files with a specific extension: *.tmp

Ignore all files in a specific directory: /build/

Ignore all files except one: *.md !README.md
Ignoring Previously Committed Files
If you want to ignore a file that has already been committed, you need to untrack it first:
git rm --cached FILENAME
